A LeetCode 34.<在排序数组中查找元素的第一个和最后一个位置>

还是不喜欢递归,我选择三指针解决问题:

function searchRange($nums, $target) {
        // 没得说..要求logN的查找直接用折半查找
        $l = 0;
        $max = count($nums)-1;
        $r = $max;
        while ($l <= $r) {
            // 先看中间指针,再分左右.比较麻烦的是刚好中间指针是值,那么就需要向左右一直找下去?
            $mid = floor(($r-$l)/2)+$l;
            // echo "{$l}:{$mid}:$r\n";
            if($nums[$mid]>$target){
                // 在左边
                $r = $mid;
            }elseif ($nums[$mid]<$target) {
                // 在右边
                $l = $mid+1;
            }else {
                // 刚好找到这种
                // 两种方式,1.简单的直接向左右扩展直到完成. 2.递归继续折半查找(时间复杂度会更低一点,但代码复杂)
                $l = $mid;
                $r = $mid;
                // echo "{$l}:{$mid}:$r\n";
                $flag = true;
                while ($flag) {
                    // echo "{$l}:{$mid}:$r\n";
                    $flag = false;
                    // 指针越界!!
                    if($l>0 && $nums[$l-1]==$target){
                        $l--;
                        $flag = true;
                    }
                    if($r<$max && $nums[$r+1]==$target){
                        $r++;
                        $flag = true;
                    }

                }
                return [$l,$r];
            }

            // 验证特别的退出条件
            if($l == $r){
                if ($nums[$l]==$target && $nums[$l]==$nums[$r]) {
                    return [$l,$r];
                }else{
                    return [-1,-1];
                }
            }
            


        }
        return [-1,-1];
    }
}

A LeetCode 36.<有效的数独>

哈希表索引算法. 唯一值得思考的就是3x3小方格,需要用整除3来代表其位置方便查询是否重复.

function isValidSudoku($board) {
    // 之前对python数据结构不够熟悉有点浪费,这次换php更简单的数据结构看看
    $rows = [];
    $cols = [];
    $squire = [];
    foreach ($board as $i => $row) {
        foreach ($row as $j => $v) {
            // 直接遍历二维数组
            if($v=='.') continue;//空白就不需要判断了
            $keyRow = "$i-$v";
            $keyCol = "$j-$v";
            $keySquire = floor($i/3).'-'.floor($j/3).'-'.$v;// 整个算法里就这个地方有点意思.
            if (in_array($keyRow,$rows) || in_array($keyCol,$cols) || in_array($keySquire,$squire)) {
                return false;
            }
            $rows[] = $keyRow;
            $cols[] = $keyCol;
            $squire[] = $keySquire;
        }
    }
    return true;
}
