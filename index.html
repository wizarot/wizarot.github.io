以后别跟我说什么只有MyISAM引擎才能用全文索引啊!

08 Mar 2017, 10:58

以后别跟我说什么只有MyISAM引擎才能用全文索引啊! 总有人在问MyISAM和InnoDB的区别时,提出只有MyISAM引擎才能使用全文索引 (Full-text Index) 老实说,我经常在自己机器的MySql中InnoDB的表上面给text加全文索引来的.而且之前我google时候,真的也在mysql官方文档里看到过InnoDB使用全文索引的介绍. 说实话,有一次我刚用完全文索引,就在被人面试的时候问过类似问题.哎,当时面试那位一脸鄙视的样子深深伤害啊! 好长时间一直都没忘记. 话说,InnoDB真的可以用全文索引,为啥中国程序员圈子里就把这个只有MyISAM引擎才能用全文索引当做铁律还一直各种面试各种问? 我也挺奇怪的. 昨天干活儿,说给InnoDB的字段加个全文索引,同事也突然提说InnoDB不能用全文索引吧? 我说刚刚才加的怎么就不行了? 同事也一脸奇怪去网上搜搜,发现官方文档确实提到可以用. 然后就愉快的用了. 今早上突然又想到这事儿,然后好好搜了一下.发现问题原来出现在这里: MySql官网文档原文 人家原文是这样写的: MySQL has support for full-text indexing and searching: A full-text index in MySQL is an index of type FULLTEXT. Full-text indexes can be used only with MyISAM tables. (In MySQL 5.6 and up, they can also be used with InnoDB tables.) Full-text indexes can be created only for CHAR, VARCHAR, or TEXT columns. A FULLTEXT index definition can be given in the CREATE TABLE statement when a table is created, or added later using ALTER TABLE or CREATE INDEX.

怎样找到自己的兴趣?

19 May 2016, 22:17

世界那么大,我们总想出去看看,工作这么多课究竟哪种才是真正符合我们兴趣的呢。之前我听说,要跟随你内心的声音,但是,抱歉,我真没听到内心有什么声音. 那么到底什么原因让我找不到真正感兴趣的工作呢?最近，公众号“心理公开课”发了篇文章说，搞不清楚“有趣”和“愉悦”这两个心理之间的区别，就是问题所在。 我们先来说说有趣和愉悦到底有什么不一样。心理学家通过实验得出这样一个结论：“有趣”是一种跟不确定性高度相关的情绪，当一件事很复杂，很新奇的时候，人们会觉得它很有趣；而“愉悦”呢，是一个和确定性高度相关的情绪，一件事越简单、越稳定，越能让人感觉到愉悦。 明白了这个，就可以解释生活中的很多现象了。比如说，我们会觉得故事里的狐狸精很有趣，但是让人愉悦的往往是良家妇女；我们觉得创业很有趣，但喜欢愉悦的人会更愿意去做公务员。所以我们是选择留在大城市打拼，还是回小城市成家，是跟自己对有趣和愉悦的偏好有关的。但是很多人不清楚自己是喜欢有趣还是喜欢愉悦，所以就会不停地做错选择。比如有的人一直以为有趣的事就会让自己很快乐，所以就不断地尝试，不断地探索。但是每次遇到困难的时候就想逃跑了，借口就是没兴趣。但是事实上，他需要的可能只是一份安稳的，让自己感到愉悦的工作。这就是我们开头说的那类人的问题所在。 文章中还提出一点，就是好的生活应该是愉悦加有趣共同构成的。当我们来到一个全新的城市、接到一个全新的任务，或者决定穿过一片森林去探险的时候，我们就会心跳加速、手心出汗，出现新奇有趣的感觉。但是如果一直这样就会压力过大，身体和心理都会不舒服。这个时候就需要愉悦来调节一下，比如在熟悉的饭店吃饭，和家人挤在沙发上看电视剧等等。这些温暖简单的事情会让我们心跳减慢，血压降低，内心变得宁静，感觉到愉悦。二者相结合，才能获得长久的幸福感。动画片为了吸引孩子的注意力，采用的就是有趣加愉悦这种模式。比如在大部分动画里，总会有一个复杂、新奇、又神秘的二号，制造趣味；同时又有一个直接、温暖、简单的主角，制造愉悦。 所以，如果你现在感觉自己不够快乐的话，不妨评估一下，自己是想要有趣更多一些，还是想要愉悦更多一些。如果你的期望和现实不相符，那就着手去改变吧。

如何找到你身边靠谱的人

17 May 2016, 21:49

我们现在无论是交朋友,谈恋爱,还是和别人一起联手创业,找到靠谱的人太重要了.作家雾满拦江在一篇文章里说,靠谱的人应该满足一下几个特点: 1.有过人的智力,能够准确预判没做过的事情的周期. 通常做好一件事情需要很长时间,不靠谱的人总觉得很快就会做好.而做事儿靠谱的人,会尽量把事情想得最困难,而且仍然放手去做的人. 2.不确定式思维,做事不求一个直线结果. 做人的人品要有确定性,这样大家才放心.但一个人做事儿的方法和方向,要有不确定性.市场变化很快,很可能开始打算做奶嘴的厂子最后去做了避孕套. 如果一个人这样想问题,那么说明这人脑子比较清醒,知道自己在做什么,而且不钻牛角尖,能够随机应变.就是个靠谱的做事儿伙伴. 3.思维周密,喜欢布闲棋. 这样的人布局具有周密性,在一个很长的周期里,肯定需要很多的资源和部件,说不定他之前做的什么布置在后来就发生巨大的作用了. 4.接受因人成事,不拘泥. 5.对人无成见,始终在路上.人总是没有确定的形态,总是要有一个发展的眼光看待别人,才有利于个人发展. 6.具有自己的实用型知识体系,化繁为简,容纳异端.这样的人爱读书还不死读书,能够善于整合观点,有自己的思维体系,能把复杂的问题简单化,并且跳出现实的条条框框.

如何面对爱人的批评

16 May 2016, 18:32

很多时候,越是和你熟悉的人,你说话越是不容易顾及对方感受. 你的爱人和家人可能就是最容易被你忽略的人. 生活中经常会随口就说对方,你很笨,你真没用这么点事儿都做不好之类的话. 你可能觉得自己说这些事无心的,也不会给对方造成什么影响,但实际上,批评就是批评,无论怎么样的批评,都是在否定对方,让人不舒服. 那么,如果你的爱人喜欢批评你,你该怎么做呢? 首先,被批评的时候你要做的第一件事儿就是不争辩和解释,而是表达自己的感受. 任何解释或争辩都无法起作用还容易使摩擦升级,你越解释对方越生气觉得你没意识到错误. 所以,你认错之后要哦把自己的感受说出来. 例如你可以成人对方说的对自己以后会小心些,或者你现在很难受,想一个人待会儿. 其次,面对一个爱批评人的爱人,你要引导对方改变习惯. 两个人在一起,实际是互相模仿的. 你们两个都在潜移默化当中互相模仿. 如果你想改变自己被批评的问题,那么好办法就是多认可和表扬对方. 当然这个方法需要时间,不可能立即生效的. 第三,如果对方做错的事情不值得表扬,我还要说表扬的话吗? 其实呢,很多事儿,虽然不值得表扬,但是却值得关心. 你的关心会影响对方的感受,这时候两人沟通起来就更容易了. 最后,有人觉得总是表扬鼓励是不是太虚伪了? 因为在我们的文化里总说好话就是恭维,不好听的才是实话. 其实呢,事情只是客观存在的,怎么看是你的事儿. 我们表扬对方,一定要表扬对方的优势. 扬长避短,关注事情的积极一面.只要你的鼓励和表扬是真心的,找准切入点,就不会显得虚伪了.

做梦能让你变聪明

15 May 2016, 22:30

很多人,包括我年轻的时候,都觉得时间不够用的时候,能够压缩睡眠时间就好了. 那么,所谓的睡眠和梦,是否是多余的呢? 世界上很多动物都会做梦,人们每天都会花一到两个小时时间做梦. 经过测试,越是聪明的动物,做梦的时间越长. 做梦居然可以让人变聪明? 为什么呢? 这是因为做梦时,大脑会激活大脑中处理白天遇到危险时的记忆区,白天学习和处理过的事情,会在晚上做梦的时候重温,并且会学习总结提炼. 这个过程可以帮助我们更高效的学习和理解. 做梦,除了可以影响智力,还能够对情绪产生很大影响. 例如,做梦可以帮我们消化抵制消极情绪,并且可以帮我们处理白天所面对的压力. 梦对我们非常重要,所以咯. 别太晚睡觉,也别为了各种原因牺牲自己的睡眠时间. 否则,会变笨啊!

不平等才是世界的常态

13 May 2016, 21:52

年轻人经常愤世嫉俗的原因是,这个世界的不公平.可是呢,其实不公平才是世界的常态.例如总体来说,贫富差距是越来越大的,越有钱的人越有钱. 可为什么世界不公平呢? 科普作家万维钢给了这么个颠覆价值观的解释: 我们知道有个80/20法则,它的意思是说,起事情决定性作用的通常是少数几个因素,而其它大部分因素都不重要. 例如企业80%的利润是20%的客户提供的;每个人80%是在20%的时间内完成的等等个. 按照这个原则来说,这个世界上大部分的财富,也是集中在极少数人的手里的. 其实这背后的原因并不复杂,因为越是有钱的人越容易赚到钱.没钱的人从一万赚到10万那是千难万难,可是有一亿的人,要赚钱到一亿零10万,那就根本不是事儿. 所以这世界的财富看上去很不平均. 实际上不只是财富,现实世界上很多东西都是遵循80/20法则分布的,录入1%的美国人拥有全美34%的财富;视频网站Netflix,到了晚上的时间段,访问他们家的视频流量站了全美35%的带宽.. 等等 那么这个道理给我们什么启示呢? 其实简单说就是应当抓住关键要点. 做产品一定要将最大的精力和注意力放在核心业务需求上面. 日常工作中,应该避免整天被一些紧急的小事儿牵着走,疲于奔命没有效率. 学习是永远是准备下一次考试的状态,不会停下来思考. 作为公司领导,却整天处理鸡毛蒜皮的小事儿而忘记看看公司的长远发展. 那么,对我们个人来说,怎么才能做到只干重要的事儿呢? 需要学会以下三点: 1.你要有自由意志,知道自己有选择的权力,不能别人让你做什么你就做什么; 2.你要学会自己判断,那些事情重要哪些不重要; 3.你还要有勇气和能力来拒绝自己不想干的事,敢和别人说 “不”. PS.工作中,千万不要陷入不停处理小事儿的陷阱. 尽量想办法做重要的事,做喜欢的事. 希望这可以对大家的工作生活有所帮助.
